# ATOM POS — Design Direction

## Brand
ATOM POS — a mobile point-of-sale app for retail garment shops.

## Colors
- Primary: `#00B5A5` (teal/green)
- Primary Dark: `#009688`
- Primary Light: `#E0F5F3`
- Background: `#F5F7FA`
- White: `#FFFFFF`
- Text Primary: `#1A1A2E`
- Text Secondary: `#6B7280`
- Border: `#E5E7EB`
- Danger / Cancel: `#EF4444`
- Success / Confirm: `#10B981`
- Warning: `#F59E0B`
- Bottom Nav BG: `#FFFFFF`
- Header BG: `#00B5A5`
- Header Text: `#FFFFFF`

## Typography
- Font: System default (React Native)
- Display: 22-28px, Bold
- Title: 18-20px, SemiBold
- Body: 14-16px, Regular
- Caption: 12px, Regular
- All teal headers use white text

## Layout
- Screen padding: 16px horizontal
- Card border radius: 12px
- Button border radius: 8px
- Input border radius: 8px
- Bottom navigation: 3 tabs (Home | POS | Reports)
- Header: Teal background, white back arrow + title

## Components
- ListItem: white card, subtle shadow, swipe actions (edit/delete/view)
- PrimaryButton: teal background, white text, full width
- CancelButton: red/outlined
- Modal: white card, rounded top corners (bottom sheet style)
- POS table: striped rows, monospace numbers
- Category tabs: horizontal scroll, teal active pill
- Product card: white, image, name, price
